# AI 감성 분석 서비스 - 프로젝트 개요

## 프로젝트명
Moodify AI

## 프로젝트 목적
사용자가 입력한 텍스트를 AI가 분석하여 감정을 긍정/부정/중립으로 분류하고,
신뢰도와 분석 이유를 제공하는 웹 기반 감성 분석 서비스 구축.

---

# 핵심 기능
- 텍스트 입력
- 감성 분석 요청
- OpenAI API 연동
- 결과 모달 출력
- 신뢰도 표시
- 분석 이유 설명

---

# 사용자 흐름
1. 사용자가 텍스트 입력
2. 감성 분석 버튼 클릭
3. 프론트엔드가 API 요청
4. 백엔드가 OpenAI API 호출
5. 분석 결과 반환
6. 결과 모달 출력
